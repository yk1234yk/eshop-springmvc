#eshop
## 项目简介
1.本项目是用于JavaEE系列方向课程中《JavaEE基础》和《软件工程实践》的演示项目。  

2.本项目的版本规划：
+ Servlet/JSP版本：1.0, 1.1, 1.2,...
+ 数据层采用Mybatis：2.0, 2.1, 2.2, 2.3,...
+ 整合SSM：3.0, 3.1, 3.2, 3.3,...
+ 启用SpringBoot：4.0, 4.1, 4.2, 4.3,...
